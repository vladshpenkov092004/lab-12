Для виконання завдання необхідно зробити fork репозиторію та клонувати його собі в дерикторію.

в файлі /css/style.css додати стилі для класу, назва якого буде аналогічна з вашим прізвищем.
Наприклад:

.vats {
    font-family: 'Segoe UI';
    color: crimson;
    font-style: italic;
}

в файлі group_103.html дописати своє ФІО та номер по списку після мого в заголовок h1 та додати йому свої стилі.

<h1 class="vats">1. Ваць Ярослав Русланович</h1>
